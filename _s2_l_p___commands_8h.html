Detailed Description

Management of S2-LP Commands.

Author
LowPower RF BU - AMG
Version
1.3.0
Date
April 12, 2017

In this module can be found all the API used to strobe commands to S2LP. Every command strobe is an SPI transaction with a specific command code.

Example:

...
S2LPCmdStrobeRx();
...
